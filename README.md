# Disneydex Movie Tracker

**Disneydex Movie Tracker** is a native Android application designed for fans of the expansive Walt Disney Company universe to explore movieographies and manage personal collections — built with Kotlin and Firebase.

## ✨ Features

| Feature               | Description                                                           |
| --------------------- | --------------------------------------------------------------------- |
| Secure Authentication | Full user account management (Login/Registration) powered by Firebase |
| Explore Universes     | Browse movies from Marvel, Disney, Pixar, Star Wars, and Avatar       |
| Movie Details         | View comprehensive lists of titles along with original release dates  |
| Collection Manager    | Track watching progress and manage physical media (DVD/Blu-ray)       |
| Community Trading Hub | Find users looking to trade or get rid of movies they own             |
| Profile Dashboard     | Dedicated space to manage personal lists and update movie statuses    |
| Real-Time Data        | Instant syncing of collection updates via Firebase Realtime Database  |


## 🛠️ Tech Stack

| Category       | Technology                               |
| -------------- | ---------------------------------------- |
| Platform       | Android Native                           |
| Language       | Kotlin                                   |
| Backend        | Firebase Realtime Database               |
| Authentication | Firebase Authentication (Email/Password) |
